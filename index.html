Important: To add addons to this repository, place the addon ZIP files inside the "repository" folder. This `index.html` file should be in the directory above the "repository" folder.

For example, if you have an addon named plugin.video.xyz, the ZIP file should be named plugin.video.xyz.zip and placed in the "repository" folder.
